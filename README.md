# FAQ компонент для 1С-Битрикс

Собственный компонент Битрикс для вывода списка вопросов и ответов из выбранного раздела инфоблока в виде аккордеона.

## Возможности

- Получение элементов FAQ из любого раздела инфоблока через D7 ORM (`ElementTable`)
- Три режима кеширования: **Автоматически**, **Включён**, **Выключен**
- Тегированный кеш — автоматически сбрасывается при изменении элементов инфоблока
- Каскадные параметры в визуальном редакторе: выбор типа инфоблока → инфоблока → раздела
- Плавная анимация аккордеона на чистом CSS (`height: auto` + `interpolate-size`)
- Делегирование событий через `BX.bindDelegate` — один обработчик на весь список

## Стек

- PHP 8+
- Bitrix D7 (ORM, Cache, TaggedCache)
- Bitrix JS (BX)
- CSS3

## Структура файлов

```
faq.list/
├── .description.php       # Название и группа компонента в визуальном редакторе
├── .parameters.php        # Параметры компонента с каскадным выбором инфоблока и раздела
├── class.php              # Логика компонента: получение данных, кеширование
└── templates/
    └── .default/
        ├── template.php   # HTML разметка
        ├── style.css      # Стили аккордеона
        └── script.js      # Переключение аккордеона через BX.bindDelegate
```

## Установка

Скопируйте папку `faq.list` в директорию компонентов:

```
/local/components/vendor/
```

## Параметры

| Параметр      | Описание                                      | По умолчанию |
|---------------|-----------------------------------------------|--------------|
| `IBLOCK_TYPE` | Тип инфоблока (каскадный выбор)               | —            |
| `IBLOCK_ID`   | Инфоблок (фильтруется по выбранному типу)     | —            |
| `SECTION_ID`  | Раздел (фильтруется по выбранному инфоблоку)  | —            |
| `CACHE_TYPE`  | Режим кеша: `A` — Авто, `Y` — Включён, `N` — Выключен | `A` |
| `CACHE_TIME`  | Время жизни кеша в секундах                   | `3600`       |

## Использование

```php
$APPLICATION->IncludeComponent(
    'vendor:faq',
    '.default',
    [
        'IBLOCK_TYPE' => 'content',
        'IBLOCK_ID'   => '1',
        'SECTION_ID'  => '5',
        'CACHE_TYPE'  => 'A',
        'CACHE_TIME'  => '3600',
    ]
);
```

## Кеширование

Компонент поддерживает три режима кеширования:

- **Автоматически (`A`)** — тегированный кеш привязанный к `iblock_id_{IBLOCK_ID}`. Автоматически сбрасывается при изменении любого элемента инфоблока.
- **Включён (`Y`)** — стандартный кеш по времени без тегов.
- **Выключен (`N`)** — без кеширования, данные запрашиваются при каждом обращении.
